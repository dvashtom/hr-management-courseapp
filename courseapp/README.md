# 🎓 אפליקציית למידה - ניהול משאבי אנוש

אפליקציית Streamlit אינטראקטיבית ללמידה למבחן בקורס ניהול משאבי אנוש.

## 🚀 התקנה והפעלה

### 1. התקנת דרישות

```bash
cd courseapp
pip install -r requirements.txt
```

### 2. חילוץ תוכן מהמצגות

```bash
python extract_content.py
```

סקריפט זה סורק את כל תיקיות הפרויקט, מחלץ טקסט מקבצי PPTX ו-DOCX, ויוצר קבצי Markdown (.md) באותן תיקיות.

### 3. הפעלת האפליקציה

```bash
streamlit run app.py --server.port 8510
```

האפליקציה תיפתח אוטומטית בדפדפן בכתובת: http://localhost:8510

## ✨ תכונות

- **📖 למידה** - צפייה בתוכן המצגות בפורמט Markdown נוח לקריאה
- **📝 תרגול** - שאלות ותשובות עם אפשרות לחשוף תשובות
- **📊 מעקב התקדמות** - סימון שיעורים שהושלמו (נשמר ב-session)
- **📱 רספונסיבי** - עובד במחשב ובנייד
- **🔤 עברית מלאה** - כולל RTL
